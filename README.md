1. Télécharge l’APK puis ouvre-le.  
2. Si Android bloque l’installation : **Paramètres → Sécurité → Installer des applis inconnues**  
   *(autoriser pour le navigateur/gestionnaire de fichiers)*.  
3. Installer → Ouvrir.
