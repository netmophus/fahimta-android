## 🧭 Installation rapide
1. Télécharge l’APK puis ouvre-le.  
2. Si Android bloque : **Paramètres → Sécurité → Installer des applis inconnues** (autoriser pour le navigateur/gestionnaire de fichiers).  
3. Installer → Ouvrir.
